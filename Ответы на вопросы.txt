1. Назначение редактора Figma
Figma используется для создания интерфейсов и прототипов, а также для совместной работы над дизайном в реальном времени.

2. Как скачать картинку из макета?
В Figma можно скачать картинку из макета, выбрав её и нажав правую кнопку мыши, затем выбрав «Экспорт». Также можно использовать панель «Экспорт» в правом нижнем углу интерфейса.

3. Назначение вкладки inspect 
Вкладка «Inspect» в Figma или в инструментах разработчика браузера позволяет разработчикам просматривать CSS-стили, размеры, шрифты и другие параметры элементов дизайна для их дальнейшего кодирования.

4. Что такое шапка сайта? Какой тег ей соответствует?
Шапка сайта — это верхняя часть веб-страницы, обычно содержащая логотип, навигацию и контактную информацию. В HTML ей соответствует тег `<header>`.

5. Для чего необходим файл normalize.css?
Файл `normalize.css` используется для устранения различий в стилях отображения элементов между разными браузерами, чтобы стилизация веб-страницы была более консистентной.

6. Вставьте ссылку для скачивания normalize.css  
https://necolas.github.io/normalize.css/

7. Пропишите команду для подключения normalize.css
<link rel="stylesheet" href="path/to/normalize.css">

8. Как подключить шрифт Roboto?
Чтобы подключить шрифт Roboto, а именно regular 400 и bold 700, нужно использовать следующий код:
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
